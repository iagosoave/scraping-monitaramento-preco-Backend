# PriceWatch - Backend

API para monitoramento de preços.

## Tecnologias

- Django 5
- Django REST Framework
- BeautifulSoup (scraping)

## Sites compatíveis

- Kabum
- Trocafone

## Rodar local

```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
